# Plan de Correction des Erreurs TypeScript Post-Migration

## Résumé
Après suppression de `app/lib/api.ts`, il reste des erreurs TypeScript dans tout le frontend dues aux :
- Références résiduelles vers `~/lib/api`
- Changement snake_case → camelCase non appliqués partout
- Modifications des types de retour du SDK (arrays direct vs { items: [] })
- Types manquants pour les mutations

---

## Erreurs par Catégorie

### 1. Routes - snake_case → camelCase dans les paths

**Fichiers concernés :**
- `app/routes/characters.tsx` : `project_id` → `projectId`
- `app/routes/project.tsx` : `project_id` → `projectId` (x7 occurrences)
- `app/routes/story.tsx` : `project_id` → `projectId`, `order_index` → `orderIndex`
- `app/routes/home.tsx` : `settings_json` → `settingsJson`

### 2. Routes - Propriétés items inexistantes

Le SDK retourne maintenant des arrays directement, pas `{ items: [...] }`.

**Fichiers concernés :**
- `app/routes/characters.tsx` : `characters.data.items` → `characters.data`
- `app/routes/home.tsx` : `projects.data?.items` → `projects.data`

### 3. Routes - Fonctions SDK mal nommées

Les noms longs générés automatiquement avec les chemins d'API complets n'existent pas.

**Fichiers concernés :**
- `app/routes/characters.tsx` : Fonctions comme `readProjectCharacterImagesApiProjectsProjectIdCharactersImagesGet` n'existent pas → utiliser `getProjectCharacterImages`
- `app/routes/home.tsx` : Même problème + `healthApiHealthGet` → `getHealth`

### 4. Routes - Import manquant useQueryClient

Dans `home.tsx`, le hook `useQueryClient` est utilisé mais pas importé.

### 5. Routes - Arguments de mutations incorrects

Certaines mutations attendent des arguments (path, body) mais reçoivent un objet direct.

**Fichiers concernés :**
- `app/routes/generation.tsx` : Appels à `create.mutate()` sans path
- `app/routes/chapter.tsx` : Mutation path incorrect

### 6. Composants - Types non exportés

Les types `Character`, `CharacterImage`, `Scene`, etc. ne sont pas exportés par le SDK de la même façon.

**Fichiers concernés :**
- `app/components/scene/CharacterImageSelector.tsx`
- `app/components/scene/SceneGenerationModal.tsx`
- `app/components/scene/SceneMangaGallery.tsx`

### 7. Composants - Propriétés snake_case

**Fichiers concernés :**
- `app/components/characters/CharacterCard.tsx` : `narrativeRole` est de type `unknown`, pas `string`
- `app/components/characters/CharacterDetailModal.tsx` : `narrative_role` → `narrativeRole`
- `app/components/characters/CharacterImageGallery.tsx` : `file_name` → `fileName`, `strategy`/`style` sont `unknown`
- `app/components/home/ProjectCard.tsx` : `root_path` → `rootPath`, `updated_at` → `updatedAt`
- `app/components/layout.tsx` : `root_path` → `rootPath`

### 8. Composants - Arguments manquants pour useMutation

**Fichiers concernés :**
- `app/components/scene/SceneGenerationModal.tsx` : `generateSceneMangaMutation()` et `previewPromptMutation()` attendent des options
- `app/components/characters/CharacterImageGenerator.tsx` : `generateCharacterImageMutation()` attend des arguments
